h2. Получение кода для хостинга

Для получения этого кода надо нажать на пункт в верхнем меню "Файлы на хостинг" и сгенерировать его при помощи мастера. Предположим, что код уже получен и вам надо вставить его на вашу любимую CMS. Общий принцип таков - получаем *llm.php* файл для вашего сайта, загружаем по фтп в специальную папку, а затем вставляем в движок сайта.

h2. Установка на Joomla 1.0.х, Joomla 1.5.x и Joostina

Проще всего устанавливать код в шаблон. Шаблон находится в директории */templates/* и представляет собой определенную директорию. В этой директории лежит файл *index.php*. Откройте его в текстовом редакторе и, например, в подвал включите выданный вам код без каких либо изменений.

h2. Установка на TextPattern

В административном интерфейсе зайдите на вкладку "Оформление" -> "Страницы". Откроется страница по умолчанию. Именно в ней и других страницах, которые были созданы вами, надо прописать ваш код, но с одним отличием. Вместо тэга *&lt;?PHP* надо написать *&lt;txp:php&gt;*, а вместо тэга *?&gt;* - *&lt;/txp:php&gt;*.

h2. Установка на Wordpress

В директории шаблона можно найти разные файлы, но лучше всего разместить код в самом низу, для этого наиболее всего подходит файл *footer.php*. Код вставляется без изменений.
