h2. Листинг сайтов

При нажатии на ссылку "Сайты" в главном меню вы попадаете в листинг уже существующих в системе сайтов. Для каждого сайта отображается его текущий PR и ТИц. Доступны следующие виды действий:
* *Просмотр страниц* - просмотр всех страниц сайта, которые удалось проиндексировать роботу, работающему в CRON.
* *Редактирование свойств сайта* - ссылка ведет на форму изменения основных данных сайта.
* *поместить в очередь индексации* - в базе данных делается отметка о том, что нужно проиндексировать сайт. Подробнее об индексации можно прочитать в разделе "CRON-задания". После выставления такой отметки индексатор при следующем запуске возьмет сайт на обработку и поиск ссылок. Помните, индексирование делается один раз. Т.е. сразу после сохранения нового сайта надо нажать на эту ссылку. Сам процесс индексирования может занять от нескольких минут, до получаса. В зависимости от объема сайта и его удаленности.
* *Обновить ТИЦ и PR морды* - пересчет PR и ТИц происходит сразу же путем нажатия на иконку справа от этих параметров. Новые данные вставляются в базу данных и видны при следующем обновлении листинга. 
* *удалить* - удаление сайта, его страниц и установленных на нем ссылок.

h2. Создание нового сайта

Для создания сайта надо нажать на ссылку *Добавить сайт* в верхнем субменю или заполнить форму внизу листинга. Любой сайт имеет следующие характеристики:
* *Домен* - URL надо писать полностью. Т.е. для домена яндекса адрес будет выглядеть так: *http://yandex.ru/* (обратите внимание на завершающий слэш ^палочка такая^ и *http://* в начале)
* *Ссылок на главной* - максимальное количество ссылок на главной странице, которое можно установить при помощи данного скрипта. В этом числе не учитываются все те ссылки, которые уже стоят на сайте и оно служит исключительно для контроля ссылок системы.
* *Ссылок на остальных* - аналогично, но для остальных страниц.
* *Разделитель ссылок* - введенные пробелы заменяются на HTML-сущность &amp;nbsp;, так как MySQL обрезает пробелы на концах строк. Максимальная длина - 255 символов.
* *CSS-класс ссылок*
* *Исключить из индексации* - список уникальных подстрок, разделенных переводом строки, при нахождении которых в адресе вновь индексируемой страницы робот удалит ее из массива индексации и не будет запрашивать. Будьте аккуратны, выставляя данный параметр.
* *FTP-доступ* - задаются следующие параметры: хост, логин, пароль и стартовая директория. При создании сайта эти параметры недоступны. После сохранения данных о сайте нужно снова войти в панель его редактирования и выставить соответствующий доступ. *Хост* - это ip-адрес или просто доменное имя, которое дает вам хостер (например, 192.168.0.1 или ftp.narod.ru, никаких приставок ftp:// или http:// быть не должно). *Логин/пароль* - непосредственный доступ. *Стартовая директория* - та директория, относительно корня ftp-аккаунта, в которой расположен сайт. Для того, что бы ее вычислить надо всего лишь открыть свой сайт в ftp-клиенте. Если вы сразу видите сайт, то оставьте ее пустой. Если для этого надо перейти по каталогам вниз - напишите этот путь. В этом случае слэш "/" должен быть в начале и конце строки.

Для задействования режима FTP-доступа его надо, во-первых, включить. Во-вторых, заполнить параметры доступа. А, в-третьих, сгенерировать файл на хостинг уже с новыми параметрами.

h2. Удаление сайта

Для удаления сайта достаточно нажать на соответствующую ссылку в листинге и подтвердить свой нелегкий выбор. Действие необратимо, хорошо подумайте, прежде чем сделать это. Если нужно временно прекратить отображение ссылок достаточно деактивировать все страницы этого сайта.

h2. Уровни вложенности страниц

Уровней всего три и различаются они по цифрам:
* 0 - главная страница
* 1 - первый уровень - это все те ссылки, которые прямо доступны с главной
* 2 - второй уровень - сумма ссылок, доступных с каждой из страниц первого уровня.

h2. Просмотр страниц

После полной индексации сайта вы сможете просмотреть те страницы, которые нашел робот. Каждая ссылка имеет следующие свойства:
* URL - адрес относительно своего сайта
* В индексе - находится ли страница в индексе поисковой системы Yandex и Google (да/нет/отменено). Значение *-1* означает, что CRON скрипт данный параметр еще не рассчитал.
* PR - параметр важности страницы для поисковой системы Google. Значение *-1* аналогично предыдущему параметру.
* УВ - уровень вложенности (0, 1 или 2)
* ВС - количество внешних ссылок
* ВС наши - количество проставленных ссылок в системе
* Активно - включена ли страница для размещения на ней ссылок и отображает ли уже расположенные ссылки (да/нет).

Слева напротив каждой ссылки вы видите чекбоксы и выбрав нужные вам страницы можно:
* Исключить страницы - они становятся не активными, т.е. на них невозможно проставить ссылки
* Активировать страницы - действие противоположно предыдущему
* Удалить страницы из таблицы - удаляются навсегда
* Не считать PR у данных страниц - проверка PR будет отменена. Данное состояние соответсвует значению *-2* в поле PR таблицы сайтов.
* Пересчитать PR у данных страниц - значения PR сбрасываются в *-1* и при следующем старте скрипта проверки PR будут обновлены.
* Отменить проверку проиндексированности + Перепроверить проиндексированность - действия, аналогичные пересчету PR, но относящиеся к проверке наличия страницы в индексе Yandex/Google

При просмотре вы можете выполнить три сервисных действия (ссылки в субменю):
* Удалить все страницы
* Пересчитать PR всех страниц - PR всех страниц будет сброшен и выставлен в -1. При следующем старте CRON-задания по измерению PR он будет обновлен.
* Пересчитать PR=0 - иногда скрипты проверки PR работают странно, поэтому в случае ошибки, дабы не перегружать их ненужными в данный момент запросами PR страницы выставляется в ноль. Лучше всего через несколько дней после проверки PR сбросить его для таких страниц и отправить на пересчет. Но это редко помогает.

h2. Просмотр ссылок системы на страницах

В листинге страниц присутствует столбец - *ВС наши*. Если ссылки присутствуют, то в нем можно увидеть их количество. Для их просмотра надо нажать на эту ссылку. При этом скрипт запросит текущее состояние данной страницы и покажет все ссылки, которые он видит.

Чуть ниже можно будет просмотреть ссылки системы. 
* Ссылку можно удалить, для этого надо нажать на текст *Убрать ссылку*, 
* А можно внести в нее коррективы. После изменения HTML-кода ссылку надо сохранить, нажав на текст *Быстрое сохранение*.
