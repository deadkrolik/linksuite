h2. Листинг проектов

Проект - это несколько ссылок, которые вы хотите проставить на какой-либо домен. Для просмотра существующих проектов надо нажать на ссылку - *Проекты* на самом верху. Возможные действия:
* *[редактировать]* - изменение HTML-кодов ссылок и названия проекта. Помните, что после установки каких-либо ссылок поменять HTML-коды, всего лишь изменив их в свойствах проекта не возможно. Эти коды нужны для формирования списка при поиске площадок. Для изменения кодов на уже проставленных ссылках надо найти их в свойствах сайта и там уже менять.
* *[просмотр ссылок]* - просмотр страниц, с которых проставлены ссылки на этот проект в системе.
* *Начать поиск площадок* - форма поиска страниц по различным критериям и ручным проставлением ссылок на отобранных страницах.

А в верхнем субменю доступно одно действие *Добавить проект* - оно вызывает форму ввода свойств нового проекта.

h2. Редактирование проекта

Каждый проект это:
* *Имя проекта* - любой набор символов, по которому вам удобно его идентифицировать. 
* *HTML-коды ссылок* - текстовое поле, в котором по одному на строку надо указать коды ссылок на ваш проект. Сами ссылки удобнее всего генерировать программой "SEO Anchor Generator":http://seogenerator.ru/. Немного въехать, а потом все будет быстро и удобно. Чем больше кодов вы сделаете, тем лучше, но помните, что длина одного кода не должна превышать 255 символов (эта длина включает в себя не только текст якоря, но и HTML-коды, нужные для формирования ссылки). Сделано это не просто так, а для того, что бы лишний раз не нагружать MySQL при большом количестве страниц и ссылок.

h2. Поиск и отбор площадок для ссылок

Поиск площадок начинается с заполнения формы параметров страниц. Чем более жестко вы зададите параметры, тем меньше страниц будет найдено. Можно регулировать следующие параметры:
* *Уровень вложенности* - чекбоксом ставим выбор на тех уровнях, которые нас интересуют.
* *PR* - поле не обязательное, можно указать диапазон, а можно только одну границу.
* *Сайты* - фильтр по сайтам.
* *Максимальное количество ВС* - максимальное количество внешних ссылок на странице. Под внешними ссылками понимаются ссылки, стоящие на сайте и не зависящие от системы. Параметр не обязательный.
* *Наличие в индексе Yandex/Google* - либо проверять, либо не проверять этот флаг у страницы.
* *Ограничить число найденных страниц с домена* - общее число страниц с каждого домена будет урезано до заданного количества.
* *Пакетное добавление* - необходимо, что бы была включена и работала крон-задача массовой расстановки ссылок. При выборе данного чекбокса каждая следующая найденная ссылка будет ставиться на каждую следующую найденную страницу со следующим по порядку якорем.
* *Только вывод страниц* - будет сгенерирован файл только со списком страниц без какой-либо навигации и прочих параметров, который можно просмотреть. Некий аналог экспорта найденных адресов страниц.
* *Только случайный выбор* - при обычном методе расстановки ссылок вы можете для каждой страницы выбрать какой-либо якорь из свойств проекта, но иногда этих якорей бывает слишком много, и они очень утяжеляют страницу (список из якорей генерируется для каждой найденной страницы). Что бы этого избежать и оставить только опцию расстановки случайного якоря - выберите этот чекбокс.

После нажатия кнопки *Найти ссылки* вам будет показан список страниц, удовлетворяющим вашим критериям. Процесс установки ссылки прост, достаточно выбрать в списке нужный якорь (или оставить "Случайный выбор") и нажать на текст *Разместить здесь ^ ^ ^* (эта кнопка распространяет свое действие на страницу сверху от нее).

h2. Просмотр ссылок

Здесь указаны все те страницы, на которых стоят ссылки на выбранный проект. Ссылку можно:

* Изменить
* Удалить
* Активировать или деактивировать

Все действия осуществляются без перезагрузки страницы при помощи AJAX.
